# Implementation Plan: Module 09 - Emotional Literacy

## Module Overview
**Target:** Дети 7-14 лет с эмоциональной дизрегуляцией, алекситимией, трудностями идентификации чувств
**Цель:** Научить распознавать, называть и понимать 8 базовых эмоций; развить эмоциональную грамотность
**Источники:** UCAM эмо-ось (8 emotions), InnerWorld002.txt, Plutchik's Wheel of Emotions adapted for children

## Core Content: UCAM Emotion Axis

### 8 Базовых Эмоций (UCAM Core):
1. **Радость (Joy)** - легкость, тепло, энергия
2. **Грусть (Sadness)** - тяжесть, слезы, потребность в утешении
3. **Гнев (Anger)** - жар, напряжение, желание защититься
4. **Страх (Fear)** - холод, сжатие, желание убежать
5. **Вина (Guilt)** - тяжесть в плечах, "я плохой"
6. **Стыд (Shame)** - желание спрятаться, "я дефектный"
7. **Отвращение (Disgust)** - "это неправильно", отторжение
8. **Удивление (Surprise)** - открытость, любопытство

## Techniques to Implement

### Technique 1: "Колесо Эмоций" (Emotion Wheel)
**Источник:** Plutchik's Wheel адаптировано для детей, UCAM 8-emotion schema

**Психологическая суть:**
- Эмоции имеют градации: раздражение → злость → ярость
- Дети часто знают только "хорошо/плохо" - расширение словаря
- Для PA детей: распознать вину vs стыд - критично для терапии
- Эмоции не "плохие/хорошие", а информационные сигналы

**Игровая механика:**
- **Квест:** "Библиотека Чувств" в Башне Эха (Tower of Echo)
- **Визуализация:** Интерактивное колесо с 8 секторами:
  - Центр: базовая эмоция (fear)
  - Средний круг: умеренная интенсивность (worried)
  - Внешний круг: высокая интенсивность (terror)
- **Интерактив:**
  - Игрок кликает на сектор → раскрывается 3-5 слов разной интенсивности
  - Пример для **Гнев**:
    ```
    Раздражение (1/5) → Злость (3/5) → Ярость (5/5)
    Недовольство → Возмущение → Бешенство
    ```
- **NPC:** Библиотекарь Слов (новый NPC) помогает подбирать точные названия

**Игровая сцена:**
```
Библиотекарь: "Что ты чувствуешь, когда тебя не слушают?"
Игрок: [Выбор эмоции]
  A. "Я в порядке" → Alexithymia flag
  B. "Мне обидно... нет, раздраженно... нет, злобно!" → точная калибровка ✓
  C. "Мне плохо" → Generic, подсказка перейти к колесу

Библиотекарь: "Давай найдём точное слово. Это лёгкая злость или сильная?"
[Открывается Emotion Wheel, сектор Гнев]
Игрок выбирает: "Раздражение" (2/5)
→ +0.05 Emotion Literacy, открыт новый диалог
```

### Technique 2: "Карта Тела Эмоций" (Emotion Body Map)
**Источник:** Research on bodily sensations in emotions (Nummenmaa et al., 2014), UCAM body-emotion link

**Психологическая суть:**
- Эмоции имеют телесный компонент (страх = сжатие в груди)
- Дети часто не осознают связь тело-эмоция → соматизация
- Раннее распознавание эмоций через телесные сигналы → лучшая регуляция
- Для PA: тело "помнит" травму, учим слышать его сигналы

**Игровая механика:**
- **Квест:** "Атлас Тела" в Башне Эха
- **Визуализация:** Силуэт тела с heat map для каждой эмоции
  - **Страх:** грудь, живот (холод, сжатие)
  - **Гнев:** лицо, руки, грудь (жар, напряжение)
  - **Грусть:** грудь, плечи (тяжесть, давление)
  - **Радость:** грудь, всё тело (лёгкость, тепло)
  - **Вина:** плечи, спина (тяжесть, сутулость)
  - **Стыд:** лицо, шея (жар, желание спрятаться)
  - **Отвращение:** живот, рот (тошнота, напряжение)
  - **Удивление:** глаза, грудь (открытость)

- **Интерактив:**
  1. Игрок выбирает эмоцию из списка
  2. Силуэт тела подсвечивается (зоны активации)
  3. Вопрос: "Где ТЫ чувствуешь это в теле?"
  4. Игрок кликает на зоны → создаётся персональная карта
  5. После 10 проверок → "Твоя Уникальная Карта Эмоций" (reward)

**Пример игровой сцены:**
```
Ситуация: Завтра контрольная, ты не готов.
NPC Эхо-Ребёнок: "Что ты чувствуешь?"
Игрок: [Выбирает: "Страх"]
Эхо-Ребёнок: "Хорошо. А ГДЕ в теле ты чувствуешь страх?"
[Появляется силуэт тела]
Игрок кликает: Живот + Грудь
Эхо-Ребёнок: "Да! Страх часто живёт в животе (холод, напряжение) и груди (сжатие).
              Теперь ты знаешь, как твоё тело сигналит о страхе."
→ +0.07 Body Awareness, +0.05 Emotion Recognition
```

**Body Map Research (визуализация в игре):**

**Школьные предметы:**
- **Биология (7-9 класс):** Вегетативная нервная система, физиология эмоций
- **ОБЖ (7-8 класс):** Реакция "бей-беги-замри", стресс и тело
- **Физкультура (5-9 класс):** Разминка, осознание мышечного напряжения
- **Окружающий мир (3-4 класс):** Как животные показывают эмоции через тело

**Локация:** **Башня Эха (Tower of Echo)**

### Technique 3: "Градусник Эмоций" (Emotion Intensity Scale)
**Источник:** DBT Emotion Regulation, UCAM intensity tracking

**Психологическая суть:**
- Эмоции имеют интенсивность (0-10 или 1-5)
- Дети склонны к all-or-nothing ("я в бешенстве!" при лёгкой злости)
- Калибровка интенсивности → лучший контроль и коммуникация
- "Мой гнев сейчас 3/10, не 10/10" → можно регулировать

**Игровая механика:**
- **Квест:** "Термометр Чувств" (доступен после Emotion Wheel)
- **Визуализация:** Вертикальная шкала 0-10 для каждой эмоции
  - 0-2: Легкая (зелёный)
  - 3-5: Умеренная (желтый)
  - 6-8: Сильная (оранжевый)
  - 9-10: Очень сильная (красный)
- **Интерактив:**
  - Игрок попадает в эмоциональную сцену
  - Выбирает эмоцию (из 8) + интенсивность (слайдер 0-10)
  - Feedback: "Точно! Это раздражение уровня 4. Ты можешь с этим работать."
  - **Прогрессия:** Чем точнее калибровка, тем больше XP

**Пример игровой сцены:**
```
Ситуация: Младший брат сломал твою игрушку.
Импульс: "Я БЕШУСЬ!" (автоматически 10/10)

[Активируется Градусник]
Эхо-Ребёнок: "Стоп. Какая это эмоция?"
Игрок: Гнев ✓
Эхо-Ребёнок: "Хорошо. А насколько сильный гнев? От 0 до 10?"
[Слайдер появляется]
Игрок двигает: 6/10
Эхо-Ребёнок: "6 из 10 - это сильная злость, но не максимум.
              Ты можешь дышать, думать, выбирать, что делать.
              Это НЕ 10/10 (где ты теряешь контроль)."
→ +0.10 Intensity Calibration, +0.05 Emotion Regulation
```

**Calibration Training:**
```javascript
{
  "Scenario 1": {
    "situation": "Учитель несправедливо поставил двойку",
    "emotion": "Anger",
    "realistic_range": "6-8/10",
    "common_error": "10/10 (overestimation)"
  },
  "Scenario 2": {
    "situation": "Друг не ответил на сообщение 2 часа",
    "emotion": "Fear/Sadness",
    "realistic_range": "2-4/10",
    "common_error": "8/10 (catastrophizing)"
  },
  "Scenario 3": {
    "situation": "Мама сказала 'ты плохо себя ведёшь'",
    "emotion": "Shame/Guilt",
    "realistic_range": "5-7/10 (PA детям)",
    "common_error": "10/10 (всё рушится)"
  }
}
```

**Артефакт:**
- **Термометр Истины** (tier 2)
- Burden: 0.04
- Эффект: +0.11 Intensity Awareness, +0.07 Proportional Response, -0.05 Catastrophizing

**Школьные предметы:**
- **Математика (5-7 класс):** Шкалы, градации, проценты
- **Физика (7-8 класс):** Измерение величин (температура, давление)
- **Литература (6-9 класс):** Градации эмоций в речи персонажей
- **География (5-6 класс):** Шкалы на картах (высота, глубина)

**Локация:** **Любая конфликтная сцена** (динамическая активация)
- VCEM: До калибровки - E=-0.7, после - E=-0.4 (снижение интенсивности)

**Коан-переход:**
```
Вход: "Что, если твоё тело говорит на языке эмоций, и ты можешь научиться его слышать?"
→
Выход: "Что, если между 'чуть-чуть' и 'очень сильно' - целый мир оттенков, и ты можешь их различать?"
```

**Reality Bridge:**
1. **Ежедневная калибровка** (вечер): "Какая была самая сильная эмоция сегодня? Сколько из 10?"
2. **Check before react**: При сильной эмоции → "Это точно 10/10 или скорее 6/10?"
3. **Словарь интенсивности**: "Раздражён (3/10)" vs "Зол (6/10)" vs "Взбешён (9/10)"

### Technique 4: "Эмоции-Информация" (Emotions as Messengers)
**Источник:** ACT + DBT, emotions as adaptive signals

**Психологическая суть:**
- Эмоции - не враги, а информаторы (гнев = нарушена граница, страх = опасность)
- Дети учат "негативные эмоции = плохо, надо убрать"
- Переформулирование: "Что эта эмоция пытается мне сказать?"
- Для PA: вина говорит "ты важен для другого", не "ты плохой"

**Игровая механика:**
- **Квест:** "Послания Сердца" (Letters from the Heart)
- **Метафора:** Каждая эмоция = посланник с письмом
- **NPC:** 8 Посланников (по одному на эмоцию):
  1. **Гонец Радости** (Joy Messenger): "Это хорошо, продолжай!"
  2. **Гонец Грусти** (Sadness Messenger): "Тебе нужна поддержка, отдых"
  3. **Гонец Гнева** (Anger Messenger): "Твоя граница нарушена, защитись"
  4. **Гонец Страха** (Fear Messenger): "Осторожно, проверь безопасность"
  5. **Гонец Вины** (Guilt Messenger): "Ты важен для другого, подумай о связи"
  6. **Гонец Стыда** (Shame Messenger): "Ты боишься отвержения, ищи принятие"
  7. **Гонец Отвращения** (Disgust Messenger): "Это нарушает твои ценности"
  8. **Гонец Удивления** (Surprise Messenger): "Новое! Исследуй!"

**Игровая сцена (пример с Виной):**
```
Ситуация: Ты сказал "нет" маме, и она обиделась.
Эмоция: Вина (intensity 7/10)

[Появляется Гонец Вины - фигура с письмом]
Гонец: "У меня письмо для тебя."
[Открывается письмо]
"Дорогой игрок,
Я - Вина. Я пришла, потому что ты важен для мамы, и она важна для тебя.
Я не говорю, что ты плохой.
Я говорю: связь под угрозой, и ты это чувствуешь.
Что ты сделаешь с этой информацией?"

Выборы:
A. "Я плохой, надо всё исправить!" → misinterpretation
B. "Связь важна. Я могу поговорить с мамой, когда она успокоится" → adaptive ✓
C. "Вина врёт, мне всё равно" → avoidance

[Если B:]
Гонец: "Да. Я выполнил свою миссию. Ты услышал послание."
→ +0.08 Emotion Wisdom, +0.06 Adaptive Response
```

**Emotion Messages Table:**
| Эмоция | Базовое послание | Адаптивная реакция |
|--------|------------------|-------------------|
| Радость | "Это хорошо для тебя!" | Продолжать, запоминать |
| Грусть | "Тебе нужна поддержка" | Искать утешение, отдых |
| Гнев | "Граница нарушена!" | Защитить себя, установить границу |
| Страх | "Возможна опасность" | Проверить ситуацию, защититься |
| Вина | "Связь под угрозой" | Восстановить связь (если здоровая) |
| Стыд | "Боюсь отвержения" | Искать принятие, self-compassion |
| Отвращение | "Это нарушает ценности" | Дистанцироваться от неправильного |
| Удивление | "Новое!" | Исследовать, учиться |

**Артефакт:**
- **Свиток Посланий** (tier 3)
- Burden: 0.05
- Эффект: +0.13 Emotion Wisdom, +0.09 Adaptive Interpretation, -0.08 Emotion Avoidance

**Школьные предметы:**
- **Литература (7-9 класс):** Мотивы героев, эмоции как драйверы действий
- **Биология (8-9 класс):** Эволюционная роль эмоций (страх = выживание)
- **История (6-8 класс):** Эмоции как двигатели исторических событий
- **Обществознание (7-9 класс):** Эмоциональный интеллект, социальная адаптация

**Локация:** **Почтовая Станция Сердца** (новая микро-локация)
- VCEM: V=+0.5, C=+0.8 (ясность), E=0.0 (любая эмоция), M=+0.7

**Коан-переход:**
```
Вход: "Что, если между 'чуть-чуть' и 'очень сильно' - целый мир оттенков?"
→
Выход: "Что, если каждая эмоция - не враг, а мудрый учитель с важным уроком?"
```

**Reality Bridge:**
1. **Диалог с эмоцией** (при сильной эмоции): "Что ты пытаешься мне сказать?"
2. **Дневник посланий** (3 раза/неделю): "Сегодня Гнев сказал мне..." (запись)
3. **Эмоция-действие**: "Если Страх говорит 'опасно', я проверю" (logic chain)

### Technique 5: "Смешанные Эмоции" (Mixed Emotions)
**Источник:** Developmental psychology, complex emotions in children

**Психологическая суть:**
- Можно чувствовать 2+ эмоции одновременно (радость + грусть при переезде)
- Дети часто думают "надо выбрать одну эмоцию"
- Для PA: "Я люблю маму И злюсь на неё" - нормально, не конфликт
- Эмоциональная сложность = зрелость

**Игровая механика:**
- **Квест:** "Перекрёстки Чувств" (Crossroads of Feelings)
- **Визуализация:** Персонаж может "держать" 2-3 эмоции одновременно
- **Сцены парадоксов:**
  - Окончание школы: Радость (новое) + Грусть (прощание)
  - День рождения после развода: Радость (праздник) + Вина (один родитель)
  - Победа в конкурсе: Радость (успех) + Страх (ответственность)
- **Механика:**
  - Игрок выбирает ДВЕ эмоции вместо одной
  - NPC Архитектор Разума: "Да! Ты можешь чувствовать оба. Это не ошибка."

**Пример сцены:**
```
Ситуация: Мама выходит замуж. У тебя будет отчим.
Традиционный выбор (ошибка):
  A. Радость (мама счастлива)
  B. Страх (что изменится?)
  C. Грусть (папа не с нами)

Новый выбор (Mixed Emotions):
  D. Радость + Страх ✓
  E. Грусть + Надежда ✓
  F. Все три: Радость + Страх + Грусть ✓

[Если выбрано D:]
Архитектор: "Да. Ты рад за маму (она счастлива) И боишься (что будет с тобой).
             Оба чувства настоящие. Оба важны. Это называется эмоциональная мудрость."
→ +0.12 Emotional Complexity, +0.08 Ambiguity Tolerance
```
